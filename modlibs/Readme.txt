Put your game mods here
ForgeGradle will deobf them and add them to classpath.
If they have coremod plugin, they will also be added to the game launch arguments for loading the coremod.
see build.gradle for the details how it works
This mod requests these mods:
mclib 2.1 (2.2 for run and debug)
metamorph 1.2.3 (1.2.5 for run and debug)
blockbuster 2.2
aperture 1.5 (1.5.1 for run and debug)
emoticons 0.7 (CompileOnly)
Optifine HD F5 (CompileOnly)
